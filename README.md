# 💎 Smart Sales Intelligence System

An AI-powered sales prediction dashboard built using Machine Learning and Streamlit.

## 🚀 Features
- 🔮 Sales Prediction using Random Forest
- 📊 Dashboard with insights
- 📈 Sales analytics visualization
- 💎 Modern glassmorphism UI

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Matplotlib

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
